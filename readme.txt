# Multi Contributors

The Multi Contributors plugin allows you to add multiple authors (contributors) to posts in WordPress. It provides a user-friendly meta box in the post editor screen for admins to select multiple contributors. The plugin adheres to WordPress coding standards, hooks, and filters.

## Features

- Add multiple authors (contributors) to posts.
- Create a meta box called "Contributors" in the post editor screen.
- Admins can select multiple contributors using checkboxes.
- Display the first name, last name, and username of contributors in the meta box.
- The current post author is checked and disabled.
- Only users with the capability to write posts are listed as contributors.
- Display the list of contributors at the front and bottom of single posts.
- Contributors are listed with their WordPress Avatar and name, linked to their author page.
- Contributors are also displayed on the author archive page.
- Adheres to WordPress coding standards and best practices.

## Installation

1. Download the plugin ZIP file.
2. Log in to your WordPress admin dashboard.
3. Go to the "Plugins" page and click on the "Add New" button.
4. Click on the "Upload Plugin" button and choose the downloaded ZIP file.
5. Click the "Install Now" button.
6. After installation, click the "Activate" button to enable the Multi Contributors plugin.

## Usage

1. Edit a post or create a new post in the WordPress admin.
2. In the post editor screen, you will find a meta box called "Contributors" on the side.
3. Select multiple contributors by checking the checkboxes next to their names.
4. The current post author will be checked and disabled.
5. Only users with the capability to write posts will be listed as contributors.
6. Save or update the post to save the selected contributors.
7. On the front end, the list of contributors will be displayed at the bottom of single posts.
8. Contributors will be listed with their avatars and names, linked to their author pages.
9. On the author archive page, posts where the user is marked as a contributor will also be listed.
